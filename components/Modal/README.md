# Модальне вікно (Modal)

Універсальний компонент модального вікна для підтвердження дій та відображення повідомлень.

## Як перевірити локально

### Варіант 1: Тестова сторінка (рекомендовано)

1. Запустіть проект:
   ```bash
   npm run dev
   ```

2. Відкрийте в браузері:
   ```
   http://localhost:3000/modal-test
   ```

3. Натисніть на кнопки для тестування різних сценаріїв модального вікна.

### Варіант 2: Інтеграція в існуючий компонент

Додайте модальне вікно в будь-який компонент:

```tsx
'use client';

import { useState } from 'react';
import Modal from '@/components/Modal';

export default function MyComponent() {
  const [isModalOpen, setIsModalOpen] = useState(false);

  return (
    <>
      <button onClick={() => setIsModalOpen(true)}>
        Відкрити модальне вікно
      </button>

      <Modal
        title="Ви точно хочете вийти?"
        message="Ми будемо сумувати за вами!"
        confirmButtonText="Вийти"
        cancelButtonText="Відмінити"
        onConfirm={() => {
          setIsModalOpen(false);
          // Ваша логіка підтвердження
        }}
        onCancel={() => setIsModalOpen(false)}
        isOpen={isModalOpen}
      />
    </>
  );
}
```

## Функціональність

✅ Закриття по кліку на backdrop (темний фон)  
✅ Закриття по кліку на кнопку "X"  
✅ Закриття по натисканню клавіші Escape  
✅ Блокування скролу body під час відкриття  
✅ Підтримка світлої/темної теми  
✅ Адаптивний дизайн (мобільні, планшети, десктопи)  
✅ Плавні анімації появи/зникнення  

## Пропси

| Проп | Тип | Обов'язковий | Опис |
|------|-----|--------------|------|
| `title` | `string` | ✅ | Заголовок модального вікна |
| `message` | `string` | ❌ | Текст повідомлення |
| `confirmButtonText` | `string` | ✅ | Текст кнопки підтвердження |
| `cancelButtonText` | `string` | ✅ | Текст кнопки скасування |
| `onConfirm` | `() => void` | ✅ | Колбек при підтвердженні |
| `onCancel` | `() => void` | ✅ | Колбек при скасуванні |
| `isOpen` | `boolean` | ❌ | Контроль видимості (за замовчуванням `true`) |

### Модальне вікно помилки

```tsx
<Modal
  title="Помилка під час збереження"
  message="Щоб зберегти статтю вам треба увійти, якщо ще немає облікового запису зареєструйтесь"
  confirmButtonText="Зареєструватись"
  cancelButtonText="Увійти"
  onConfirm={() => router.push('/auth/register')}
  onCancel={() => router.push('/auth/login')}

```

## Примітка про помилки на localhost:3000

**Це нормально!** Помилки при реєстрації/вході на `localhost:3000` виникають, тому що:

1. **Бекенд не запущений локально** - проект використовує зовнішній бекенд на `https://travel-fs116-teamproject-backend.onrender.com`
2. **Модальне вікно працює без бекенду** - це UI компонент, який не потребує підключення до API
3. **Для тестування модального вікна бекенд не потрібен** - використовуйте тестову сторінку `/modal-test`

Якщо потрібно протестувати повний функціонал з бекендом:
- Запустіть бекенд локально (якщо він доступний)
- Або використовуйте задеплоєну версію на продакшні
